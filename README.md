## Featured Project

### [JOLT — Job Opportunity Logic Tracker](https://github.com/RafaelAlbaWebify/jolt-job-tracker)

Local job-search decision assistant built with React and FastAPI.

JOLT parses user-provided job text or page text, applies configurable fit rules, explains Apply / Maybe / Discard / Manual Review decisions, exports results to JSON / CSV / XLSX, and tracks reviewed jobs locally.

It is not a scraper and not an auto-apply bot. Browser automation is disabled/experimental; the portfolio value is in the parser, rule profiles, explainable decisions, export package, and local tracker.
